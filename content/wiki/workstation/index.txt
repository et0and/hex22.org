text: I currently use a 2015 Macbook Pro 13" with 8GB of RAM and 2.7Ghz i5 processor.

![Screenshot of my desktop](/content/wiki/workstation/desktop.png)

### Graphical

- Browser: Safari / [Firefox](https://mozilla.org/firefox) / [Beaker](https://beakerbrowser.com/)
- Terminal: [Hyper](https://hyper.is/)
- Text Editor: [Atom](https://atom.io/) / [Left](https://hundredrabbits.itch.io/left)
- Design: [Figma](https://figma.com)
- Window Manager: [Spectacle](https://www.spectacleapp.com/)
- Audio Workstation: [Ableton Live](https://ableton.com/live)
- Music: [Spotify](https://spotify.com/)
- Video Player: [IINA](https://lhc70000.github.io/iina/)
- Calculator: [Numi](https://numi.io/)
- Chat: [Telegram](https://telegram.org)

### Command-Line

- Text Editor: [micro](https://github.com/zyedidia/micro)
- npm publish: [np](https://github.com/sindresorhus)
- Git: see [dotfiles](https://github.com/kodedninja/dotfiles)
- Node Version Manager: [n](https://github.com/tj/n)
- Domains: [now](https://github.com/zeit/now-cli)
- Localtunnel: [localtunnel](https://github.com/localtunnel/localtunnel)
- HTTP Server: [http-server](https://npmjs.com/package/http-server)
- Image Processing: [korkut](https://github.com/oguzhaninan/korkut)

For the libraries I usually use, go to [this repo](https://github.com/kodedninja/weapons).
